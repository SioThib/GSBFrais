Pour l'actualisation des tables FicheForfait, LignesFraisForfait et LignesFraisHorsForfait, il faut
- Avoir installé la base et les tables FraisForfait, Visiteur et Etat
- Copier le dossier dans le répertoire de publication
- Ouvrir avec un éditeur le fichier majGsb_main.php
- remplacer les valeurs de la connexion à la base avec vos paramètres
- lancer l'interprétation du script avec le navigateur
